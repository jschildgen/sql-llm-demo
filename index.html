About SQL Invaders

SQL Invaders is an exciting space-themed game that allows players to control the action using SQL commands and their voice.

The game can be played on PC browsers or on the Meta Quest 3 VR headset using the Wolvic browser.

  • Developer: Johannes Schildgen
  • Release Date: 2024
